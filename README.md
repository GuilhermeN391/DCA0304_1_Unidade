# DCA0304 - 1ª Unidade

Trabalho desenvolvido para a primeira unidade da disciplina de Métodos Computacionais Em Engenharia - DCA0304
  
## Contextualização

- Neste repositório estão dispostos os códigos e algumas etapas do relatório feito para a primeira parte da 1ª Avaliação da Disciplina, para facilitar a apresentação do que foi realizado na atividade;
- Ele possui um programa para execução de resolução de sistemas não lineares a partir do método Gauss-Siedel, dado o contexto de uma matriz de potenciais 5x3;
- As matrizes adicionadas em .txt emulam o sistema apresentados e os arquivos em "Soluções" foram os resultados obtivos na execução do método.
